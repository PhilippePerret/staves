Retourne les étapes sélectionnées. Permet de ne jouer qu'une partie du code. Moins les étapes de préambule éventuelles

Notes

  • Mais l'opération n'est pas aussi simple que ça : il faut jouer les code qui gèrent les portées et autres réglages par défaut. Donc l'opération se fait en deux temps :
    1. On prend le code sélectionné pour connaitre l'offset et le contenu
    2. On passe en revue l'intégralité du code en conservant les lignes qui doivent être jouées.
  • On mémorise les informations pour pouvoir rejouer la même sélection
  • La liste renvoyée est expurgée des pas de "préambule" ou peuvent être définies des préférences.
