Provided by the Scale.js module.

Defined in /Users/philippeperret/Sites/staves/javascript/Notes/Scale.js:4

Objet construisant une gamme

Parameters:

  • scale String

    La gamme à construire, exprimée par une note et une altération

  • params Object

    Les paramètres optionnels

    • staff Number

      Indice de la portée. Défaut : la portée courante.

    • type String

      Le type de gamme parmi :

                                      * 'MAJ' (défaut = gamme majeure), 
                                * 'min_h' (mineure harmonique), 
                                * 'min_ma' (mineure mélodique ascendante)
                                * 'min_md' (mineur mélodique descendante)
    • offset Number

      Le décalage pixel entre chaque note. Par défault le décalage optimal en fonction des altérations

    • octave Number

      Octave de la note de départ. Par défaut, la meilleure position par rapport à la portée et la note, pour produire le moins de lignes supplémentaires possible.

    • asc Boolean

      Ascendante ou descendante (true par défaut)

    • from String

      La note de laquelle on doit partir, si c'est une autre note que la fondamentale.

    • for Number

      Le nombre de notes à écrire (8 par défaut)

Returns:

Array:

un Array des instances notes dont les méthodes et propriétés ont été étendus par METHODES_GROUPNOTES et PROPERTIES_GROUPNOTES.
