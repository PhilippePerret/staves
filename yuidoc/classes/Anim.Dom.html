Defined in /Users/philippeperret/Sites/staves/javascript/Anim/>Dom_Doublage.js:140

Méthode qui efface le doublage d'indice +idoub+. En fait, ça n'est pas le dernier, mais l'avant dernier, car :

  • Un doublage est en cours d'affichage
  • Le doublage précédent est en cours d'écho
  • Le doublage d'avant peut être détruit complètement.

Parameters:

  • idoub Number

    Indice du DIV doublage à détruire

      Si false => doublage, si true => caption

    • wait Boolean
