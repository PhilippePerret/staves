REGEXP_NOTE Class

Defined in: /Users/philippeperret/Sites/staves/javascript/Notes/Note.js:87
Module: Note

Le motif régulier pour analyser la note

  • [optionnel] portée désignée par 1 ou 2 chiffres suivis de ":"
  • [mandatory] Le nom de la note (une lettre de "a" à "g")
  • [optionnel] L'altération (une lettre parmi "d", "b", "x", "t", "z")
  • [optionnel] Le signe "-" de l'octave
  • [optionnel] l'octave (un nombre de 0 à 8) - il est optionnel dans un motif

moveTo

(
  • new_note
  • params
)

Defined in /Users/philippeperret/Sites/staves/javascript/Notes/Note.js:269

Déplace la note à la hauteur +hauteur+ (noter qu'il s'agit en fait d'un changement de note. On l'utilise aussi s'il faut simplement ajouter une altération).

Notes

  • Le déplacement a été simplifié : la note change de couleur et se déplace de façon rectiligne en passant au-dessus des autres
  • L'option prise, par rapport à l'altération, est de la faire disparaitre et ré-apparaitre à la nouvelle position. Sinon c'est trop compliqué (et pas nécessaire) de la faire se déplacer aussi. Cf. la méthode analyse_note qui s'en charge le cas échéant.

Parameters:

  • new_note String

    La nouvelle note à atteindre

  • params Object

    Les paramètres optionnels

    • no_exergue Boolean

      Si TRUE, ne met pas la note en exergue

    • complete Function | False

      La méthode pour suivre. NEXT_STEP PAR DÉFAUT (méthode "terminale")
